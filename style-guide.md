# Front-end Style Guide

## Layout

The designs were created for the following widths:

- Mobile: 375px
- Desktop: 1440px

## Colors

- **Background Color**: `#141414`
- **Card Background Color**: `#1f1f1f`
- **Text Color**: `#ffffff`
- **Active Button Color**: `#c4f82a`
- **Inactive Button Color**: `#333333`

**Color References:**

- **Green**: `hsl(75, 94%, 57%)`
- **White**: `hsl(0, 0%, 100%)`
- **Grey 700**: `hsl(0, 0%, 20%)`
- **Grey 800**: `hsl(0, 0%, 12%)`
- **Grey 900**: `hsl(0, 0%, 8%)`

## Typography

### Body Copy

- **Font Size (Paragraph)**: `14px`
